# 🫁 Detecting COVID-19 With Chest X-Ray Using PyTorch

## 📖 Overview
This project applies deep learning and computer vision to the healthcare domain by building an image classification model to detect COVID-19 from chest X-ray scans. The notebook walks through the entire pipeline, from loading and visualizing medical image data to applying data transformations, and finally leveraging transfer learning with a pre-trained ResNet18 model in PyTorch.

## 🎯 Objective
The primary objective is to develop a highly accurate deep learning vision model to automate the detection of COVID-19 using chest X-ray imagery. By utilizing PyTorch and the ResNet18 architecture, the project aims to demonstrate how transfer learning can be applied to complex medical image classification tasks to assist in clinical diagnostics.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Deep Learning Framework:** PyTorch, Torchvision (ResNet18)
* **Data Manipulation & Visualization:** NumPy, Matplotlib, Seaborn

## 🧠 Key Learnings & Skills Demonstrated
* **Problem Formulation:** Understanding the clinical importance of automated COVID-19 detection using medical imagery.
* **Exploratory Data Visualization:** Importing and visualizing chest X-ray images to understand the dataset's structure and clinical variations.
* **Image Pre-processing & Transformation:** Applying PyTorch transforms, including resizing, cropping, and normalization, to prepare the image data for neural network ingestion.
* **Transfer Learning:** Utilizing the pre-trained ResNet18 architecture to extract features and adapt it for binary medical image classification.
* **Model Training:** Setting up loss functions, optimizers, and data loaders to train the deep learning model across multiple epochs.
* **Performance Evaluation:** Testing the trained ResNet18 model and interpreting evaluation metrics to assess its diagnostic accuracy.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Detecting-COVID-19-With-Chest-XRay-Using-PyTorch.git

2. Navigate into the downloaded project folder:
   cd Detecting-COVID-19-With-Chest-XRay-Using-PyTorch

3. Install the required dependencies:
   pip install torch torchvision numpy matplotlib seaborn jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to visualize the X-rays and train the ResNet18 model.
